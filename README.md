# Dailies

A fun, interactive Habit Tracker built with Android, Jetpack Compose, and Material You dynamic colors.

## Features
- **Dynamic Theming:** Uses Android's Material You to theme the app based on your wallpaper.
- **Multiple Completions:** Some habits take more than one action! Click the interactive bubbles to mark your progress.
- **Widgets:** A home screen Glance widget to remind you of your daily goals without opening the app.
- **Notifications:** A daily morning reminder to ensure you never break your streak!

## How to Install (No Code Required!)
Thanks to GitHub Actions, you don't need Android Studio to build this app.
1. Fork or push this repository to GitHub.
2. Go to the **Actions** tab in your GitHub repository.
3. Click on the latest workflow run.
4. Scroll down to **Artifacts** and download `app-debug.apk`.
5. Send the APK to your Android phone, click on it, and install!
